js技术总结—-BOM

导语

DOM是文档对象模型
BOM(Browser Object Model)是浏览器对象模型；
BOM 是关于浏览器的方法，属性，事件。

常用BOM方法

  1. window.open()；打开新的页面
    参数：
    (1)代表要打开的新网页的地址；
    (2)打开方式，是在本页面打开还是另外一个页面打开：
    _self 在本窗口打开；
    _black 在新窗口打开；
    _iframName 在一个网页里面嵌套另一个网页
    (3)打开新窗口的位置信息。

  2. window.close;关闭新的页面
    不同的浏览器的表象不一样，有的关闭的是它本身的页面，有的关闭的是新打开的页面。

  3. window.location—-地址栏信息；
    字符串版的地址栏信息；

  4. BOM的相关尺寸；

  • 可视区的宽和高：
    window.innerWidth
    window.innerHeight
    BOM方法可视区的方法兼容不太好，不兼容低版本的浏览器
  • scrollTop 被卷取的页面高度
    scrollLeft 被卷取的页面宽度
  • scrollWidth 实际内容的宽度，不含边框
    scrollHeight 实际内容的高度，不含边框
  • 屏幕的宽和高
    window.screen.height 测量屏幕的高度，不受窗口大小的影响；
    window.screen.width 测量屏幕的长度，不受窗口大小的影响；
  • 可利用的宽高
    window.screen.availHeight
    window.screen.availWidth
  • document相关的宽和高
    document.body.clientWidth 可利用的宽度，受窗口大小的影响；
    document.body.clientHeight可利用的高度，受窗口大小的影响；
  • 获取元素的宽高
    document.body.offsetWidth 获取元素的宽；
    document.body.offsetHeight获取元素的高。

滚动条的距离

  • DOM：(纵向滚动条)
    chrom document.body.scrollTop;
    其他浏览器 document.documentElement.scrollTop;
  • BOM (不考虑低版本可是用这个)
    window.pageXoffset 横向滚动条
    window.pageYoffset 纵向滚动条
  • 设置滚动条的距离
    window.scrollTo(x,y)
    这个方法是全兼容，两个参数必须同时出现。

滚轮事件

  • window.onscroll();
    当滚动条滚动的时候触发这个方法；
    它会按照一定的频率来触发，不同的浏览器频率是不一样的，滚动的频率也是不一样的；
    并不是滚动一下就是一像素，有的浏览器频率高，有的频率低；
  • window.onresize();
    当窗口改变大小的时候触发这个方法。
